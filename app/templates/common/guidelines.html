Guidelines

  • Upload the answer sheet in the form of an image.
  • Make sure the image is clear and the text is readable.
  • Make sure the image is not too large.
  • Make sure the image is not too small.
  • Make sure the image is not too blurry.
  • Make sure the image is not too bright.
  • Make sure the image is not too dark.
  • Make sure the image is not too tilted.
  • Make sure the image is not too skewed.
  • Make sure the image is not too rotated.
  • Make sure the image is not too noisy.
  • Make sure the image is not too compressed.
  • Make sure the image is not too stretched.
  • Make sure the image is not too squeezed.
  • Make sure the image is not too distorted.
  • Make sure the image is not too pixelated.
  • Make sure the image is not too grainy.
  • Make sure the image is not too overexposed.
  • Make sure the image is not too underexposed.
  • Make sure the image is not too saturated.
  • Make sure the image is not too desaturated.
  • Make sure the image is not too contrasty.
  • Make sure the image is not too brightened.
  • Make sure the image is not too darkened.
  • Make sure the image is not too sharpened.
  • Make sure the image is not too blurred.
  • Make sure the image is not too filtered.
